# Osztályozó algoritmusokat összehasonlító python szkript

A notebook futtatásához a szokásos data scientist python könyvtárakra van szükség: `jupyter notebook`, `numpy`, `pandas`, `sklearn`, `matplotlib`, `seaborn`.
